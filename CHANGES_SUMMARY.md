## 6. آلية العمل

### عملية الإرسال:

```
المستخدم يملأ النموذج
         ↓
النموذج يتحقق من البيانات المطلوبة
         ↓
يتم الإرسال عبر API (/api/submissions)
         ↓
النظام يحفظ البيانات في JSON
         ↓
يعرض رسالة تأكيد للمستخدم
         ↓
تظهر الطلبات في لوحة التحكم الإدارية
```
